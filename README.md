## requirements

- zsh-completions
- zsh-syntax-highlighting
- anyframe
- peco / percol / fzf
- ag

## install

```
mkdir -p ~/.cache/shell
mkdir -p ~/.zsh/antigen/
curl -L https://raw.githubusercontent.com/zsh-users/antigen/master/antigen.zsh > ~/.zsh/antigen/antigen.zsh
ln -s ~/dot-zsh-files/.zshrc ~/.zshrc
source ~/.zshrc
```
